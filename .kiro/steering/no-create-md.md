# No crear archivos Markdown sin autorización

- NO crear archivos `.md` nuevos a menos que el usuario lo autorice explícitamente.
- NO generar documentación adicional sin aprobación.
- Si se necesita documentar algo, preguntar primero antes de crear el archivo.
- **Excepción**: Sí se pueden crear archivos `.md` dentro del directorio `.kiro/` (specs, steering, etc.).
